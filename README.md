## 👨‍🏫 프로젝트 개요

비트공방은 공예 작가와 공예 애호가가 작품, 재료, 기법, 제작 과정을 함께 공유하고 탐색할 수 있는 공예 전문 커뮤니티 웹 서비스입니다.

기존 SNS처럼 완성작 이미지만 빠르게 소비하는 방식이 아니라, 작품이 만들어지는 과정과 사용된 재료, 기법, 날짜별 작업 기록까지 함께 보여주는 것을 목표로 합니다.

✔️ 팀 구성 및 역할  

* **팀원명** 🖥️ 회원가입 / 로그인 / 마이페이지
* **팀원명** 🖥️ 공예 프로젝트 CRUD / 프로젝트 상세 페이지
* **팀원명** 🖥️ 날짜별 제작 로그 / 이미지 업로드
* **팀원명** 🖥️ 커뮤니티 게시판 / 댓글 기능
* **팀원명** 🖥️ 검색 / 필터 / 좋아요 / 저장 기능
* **팀원명** 🖥️ UI 디자인 / 프론트엔드 화면 구현
* **팀원명** 🖥️ DB 설계 / ERD / 테스트 및 배포

✔️ 전체일정

| 주차 | 주요 작업 |
|---|---|
| 1주차 | 기획 정리, 화면 설계, Git/GitHub 환경 구성 |
| 2주차 | HTML/CSS/JavaScript 기반 주요 화면 구현 |
| 3주차 | DB 설계, ERD 작성, Spring MVC 프로젝트 구조 설정 |
| 4주차 | 회원가입, 로그인, 로그아웃, 마이페이지 구현 |
| 5주차 | 커뮤니티 게시판 CRUD, 댓글 기능 구현 |
| 6주차 | 공예 프로젝트 CRUD, 날짜별 제작 로그 구현 |
| 7주차 | 검색/필터, 좋아요/저장, 통합 테스트, 배포 및 발표 준비 |
